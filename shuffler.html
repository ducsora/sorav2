Cách hoạt động

Công cụ cắt video gốc thành nhiều đoạn nhỏ → xáo trộn / sắp xếp lại theo template → tạo ra N biến thể khác nhau. Hoàn toàn xử lý trong trình duyệt — không cần upload server.

Định dạng xuất: Chạy qua start-server.bat để xuất .mp4. Mở file:// trực tiếp sẽ fallback sang .webm.

⚠️ Cần chạy Local Server để xuất MP4

FFmpeg.wasm yêu cầu HTTP server để tải file .wasm. Hiện bạn đang mở qua file://.
Cách khắc phục: Chạy start-server.bat → mở http://localhost:3001/shuffler.html. Tool sẽ tự fallback WebM nếu không có server.
